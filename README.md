## 🏁 Example Profile Configuration

Example `.skylark.yml` for project rules and custom events:

    settings:
      tracked_repos:
        - name: my-awesome-service
          triggers:
            - type: milestone
              at: 10_000
              action: notify
              channels: ['slack', 'email']
          analytics: true
      ai_assistant:
        provider: OpenAI
        language: 'en,es'
      dashboard:
        theme: 'auto'
        alerts: true
        aggregation: true
      privacy:
        data_sharing: false
        retention: 180 # days

Want to automate? Try:

    $ skylark notify --trigger all --proto slack,email
